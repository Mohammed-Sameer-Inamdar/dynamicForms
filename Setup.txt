# Dynamic_forms_react 
    This is the front part built using react

# dynamicFormsExpress
    This is the backend part built using node and express

# Database 
    MogoDB atlas of free tier is used.

# Front end setup
    1. Open prject in vs code or any other editor
    2. open cmd or terminal
    3. run "npm install" or "yarn install"
    4. run "npm start" 

# backend setup
    1. Open prject in vs code or any other editor 
    2. open cmd or terminal
    3. run "npm install"
    4. run "nodemon index.js"
